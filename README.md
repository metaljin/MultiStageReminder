# 分段提醒器 (Multi-Stage Reminder)

一款支持多阶段、自定义铃声与震动的定时提醒应用，适用于上课铃、番茄钟、健身计划等需要分时段提醒的场景。  
每个模板可独立设置循环次数、总铃声开关、震动开关及响铃时长，阶段内可自定义时长与铃声。

## 功能特性

- **多阶段提醒**：一个模板可包含多个阶段，顺序执行，每个阶段独立设置时长（分钟）与铃声。
- **模板整体设置**：
  - **循环次数**：模板整体重复执行次数。
  - **铃声开关**：是否播放铃声（若关闭则仅震动或静音）。
  - **震动开关**：是否在提醒时震动。
  - **响铃时长**：铃声播放的最大时长（秒），避免长响干扰。
- **灵活铃声选择**：
  - 支持本地音频文件（通过文件管理器选择）
- **后台运行**：使用前台服务，锁屏/应用退出后仍可正常提醒。
- **简洁控制**：运行时显示当前阶段、剩余时间，支持停止与重新开始。



## 技术栈

- **语言**：Kotlin
- **UI**：Jetpack Compose（Material 3）
- **数据持久化**：Room 2.6.1 + Kotlin 协程
- **后台服务**：Foreground Service + Handler 计时
- **音频播放**：MediaPlayer / RingtoneManager
- **最小 SDK**：API 26（Android 8.0）
- **目标 SDK**：API 34（Android 14）

## 使用说明

### 创建模板
1. 主界面点击“+”按钮
2. 输入模板名称、循环次数（正整数）、铃声开关、震动开关、响铃时长（秒）
3. 添加阶段：为每个阶段设置时长（分钟）与铃声（可分别选择）
4. 保存模板

### 开始提醒
- 在模板列表点击模板右侧的“启动任务”按钮，应用将进入计时界面并开始倒计时。
- 计时过程中可点击“停止任务”、“暂停”、“恢复”、“跳过”控制。

### 运行时权限
- **通知权限**（Android 13+）：首次启动时会请求，用于显示前台服务通知。
- **音频读取权限**：当选择本地音频文件作为铃声时，需授予 `READ_MEDIA_AUDIO`（Android 13+）或 `READ_EXTERNAL_STORAGE`（旧版本）。

## 注意事项

- 为保证后台提醒稳定，应用会运行一个前台服务，状态栏会显示持续通知（可下拉查看进度）。
- 设备开启“省电模式”或“休眠优化”时，可能会影响计时精度，建议将应用加入白名单。
- 铃声播放时，若同时有多个提醒触发（极少情况），后一个会覆盖前一个。
